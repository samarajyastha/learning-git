# Learning GIT

## GIT

- Version control system (VCS)
- Linus Torvalds build it.

### Git commands

1. Initialize git
`git init`

2. Status
`git status`

3. Track files
`git add <filepath/filename>`
`git add .` (track all)

4. Commit changes
`git commit -m "<message>"`

5. Create a branch
